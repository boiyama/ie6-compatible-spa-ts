# ie6-compatible-spa-ts
IE6-compatible SPA example in TypeScript

## DEMO
https://boiyaa-ie6-compatible-spa.appspot.com

## Web

### Development

```sh
$ cd web
$ npm i
$ npm run dev
```

### Deployment

```sh
$ npm run build
$ gcloud app deploy
```

## API

### Development

```sh
$ cd api
$ npm i
$ npm run dev
```

### Deployment

```sh
$ gcloud app deploy
```
